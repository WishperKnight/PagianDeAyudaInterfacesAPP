UserAccountDAO

La clase `UserAccountDAO` es fundamental para la gestión de usuarios en la aplicación, siguiendo el modelo CRUD (Crear, Leer, Actualizar, Eliminar).

Este método permite agregar un nuevo usuario a la base de datos. En la imagen, puedes ver el código que implementa esta funcionalidad, insertando un nuevo usuario con sus datos correspondientes.

El método `comprobarUser` se encarga de verificar las credenciales de un usuario. En la imagen, se muestra el código asociado, donde se comprueba si las credenciales ingresadas coinciden con las almacenadas en la base de datos.

Este método, denominado `validateUser`, verifica si un nombre de usuario ya existe en la base de datos. El código presentado en la imagen realiza la consulta necesaria para determinar la existencia del usuario proporcionado.
